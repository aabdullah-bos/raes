# Simple Web Calculator PRD

## Purpose
Build a simple web-based calculator that allows a user to perform basic arithmetic operations in the browser.

## Target User
A user who wants a fast, clear calculator for basic math.

## Core Features
- Enter numbers using on-screen buttons
- Perform addition, subtraction, multiplication, and division
- See the current expression or input
- See the calculated result
- Clear the current input
- Reset the calculator state

## Success Criteria
- User can complete a basic calculation entirely through the UI
- Calculator updates predictably based on button presses
- Invalid states are handled in a consistent way
- UI is simple and understandable

## Out of Scope
- Scientific functions
- Keyboard shortcuts
- Calculation history
- Authentication
- Backend persistence

## Experience Intent
The calculator should feel:
- immediate
- clear
- reliable
- simple, not playful
